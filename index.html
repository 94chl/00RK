전체 최단 루트
  • 사용법 : 아이템 추가/제거 후, "계산하기" 버튼 클릭
  • 루트 탐색 기준 : 현재 위치에서 인접한 지역 중 필요한 드랍템이 나오는 지역을 다음 목표로 탐색
    (하이퍼루프 지역은 모든 지역을 탐색)
  • 우선 지역 지정 : 원하는 지역을 미리 클릭후 "계산하기" 버튼 클릭
  • 주의1) 루트 경로가 6지역 이상일 경우 탐색 시간이 길어짐
  • 주의2) 가방의 소지 한도를 계산하지 않기 때문에, 하단의 가방 기능으로 직접 시연 권장
  • 주의3) 필요한 드랍템이 공통 드랍템 뿐일 경우, 추천 경로를 표시하지 않음
